```bash
# Conventional Commits
<type>(<scope>): <subject>

# 类型
feat:     新功能
fix:      Bug 修复
docs:     文档更新
style:    格式调整
refactor: 重构
test:     测试
chore:    构建/工具

# 示例
feat(channels): add WhatsApp group mention support
fix(config): resolve env substitution edge case
docs(gateway): update authentication guide
```
